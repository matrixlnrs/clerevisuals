# ClereVisuals

This project is a portfolio website built with Angular, designed to showcase my work to brands interested in collaborating with me.

## Tech Stack
Angular
TypeScript
HTML / CSS

## Project Structure (simplified)
`src/
├── app/
│   ├── navbar/
│   ├── landing/
│   ├── gallery/
│   └── app.component.*
└── styles.css`

## Getting Started
Install dependencies
`npm install`
Run the project
`ng serve`
Then open:
`http://localhost:4200`
