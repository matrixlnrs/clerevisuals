# ClereVisuals

This project is a portfolio website built with Angular, designed to showcase my work to brands interested in collaborating with me.

## About
I’m a director and photographer focused on creating cinematic, emotion-driven content. This website serves as a curated space where brands can explore my work, understand my creative approach, and envision potential collaborations.

## Tech Stack
- Angular
- TypeScript
- HTML / CSS

## Project Structure
```src/
src/
├── app/
│   ├── navbar/
│   ├── landing/
│   ├── gallery/
│   └── app.component.*
└── styles.c
```

## Getting Started
Install dependencies
`npm install`
Run the project
`ng serve`
Then open:
`http://localhost:4200`
